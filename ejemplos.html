Sistemas de Lazo Abierto
La señal de salida no realimenta al controlador. El sistema actúa según una consigna fija (tiempo, posición, potencia) sin verificar si el resultado fue el esperado. En presencia de perturbaciones, el error no se corrige automáticamente.
Sin sensor · Sin comparador · Sin retroalimentación
🔁
Sistemas de Lazo Cerrado
La señal de salida realimenta al controlador. El sensor capta la variable controlada, el comparador detecta el error respecto a la referencia y el controlador corrige la acción. Ante perturbaciones, el sistema se autoregula.
Con sensor · Con comparador · Con retroalimentación negativa
Característica 🔓 Lazo Abierto 🔁 Lazo Cerrado
¿Hay sensor?NoSí — mide la variable de salida
¿Hay retroalimentación?NoSí — la salida vuelve al comparador
¿Hay comparador?NoSí — calcula la señal de error
¿Reacciona a perturbaciones?No — el error persisteSí — se autoregula
Rombo de decisión en el diagramaGeneralmente noSí — obligatorio
Complejidad del sistemaMenor — más económicoMayor — más preciso y robusto
